# ChatGPT 工具推荐文章归档说明

这篇文章最初发布于 2023 年，主要介绍当时可用的第三方 ChatGPT 站点。由于第三方服务的可用性、隐私政策和收费方式都可能变化，原文已不再作为当前使用建议。

如果你希望系统学习如何使用大模型，请阅读：

- [如何使用大模型提升学习和求职效率](./如何使用大模型提升学习和求职效率.md)
- [大模型使用专题：从高质量提问到个人知识库](./大模型使用/README.md)

使用任何在线工具前，请注意：

1. 优先选择来源明确、隐私政策清晰的服务。
2. 不要上传身份证号、手机号、真实密钥、未公开代码和公司内部资料。
3. 对技术结论、招聘规则和实时信息进行核验。
4. 将大模型作为辅助工具，而不是替代独立思考。
