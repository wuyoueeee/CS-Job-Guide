# 30 天求职训练营

这个训练营用于把仓库内容转化为连续行动。每天只做一件最关键的事，并留下可检查的交付物。

## 使用规则

1. 每天最多投入 2 到 4 小时，保持连续性。
2. 每天必须产出一个文件、表格、清单或讲解稿。
3. 第 7、14、21、30 天做复盘，不继续盲目前进。
4. 如果已经在面试中，可以跳过前置内容，直接从卡点对应天数开始。
