# 大模型使用专题

![大模型使用专题封面](./images/大模型使用专题封面.png)

大模型不是答案机器，更像一位随时在线的助教、面试官和讨论伙伴。用得好，它可以帮助你更快地发现问题、组织知识和验证理解；用得不好，它也可能让你在看似高效的复制粘贴中失去判断力。

这个专题面向正在准备实习、校招和职业发展的计算机专业学生。建议先阅读总览，再根据当前任务选择对应文章。

## 使用原则

1. 先明确问题，再向模型提问。
2. 对重要结论进行验证，不盲目复制答案。
3. 不上传敏感数据、私有代码和真实密钥。
4. 把模型用于增强思考，而不是替代思考。
