# 面经脱敏规范

真实面经很有价值，但必须保护个人和企业敏感信息。

## 一、必须删除或模糊的信息

| 信息 | 处理方式 |
| --- | --- |
| 姓名、手机号、邮箱、微信 | 删除 |
| 学号、身份证、准考证 | 删除 |
| 面试官姓名和联系方式 | 删除 |
| 内部系统截图 | 不上传 |
| 未公开题库、文档、代码 | 不上传 |
| 具体薪资截图 | 可改成区间或描述 |
| 学校和专业 | 可按需要模糊为 985 / 211 / 双非 / 硕士 |

## 二、推荐保留的信息

- 公司和岗位。
- 批次和时间范围。
- 面试轮次。
- 问题类型。
- 项目追问方向。
- 自己的复盘和准备建议。

## 三、面经格式建议

```markdown
# 公司 + 岗位 + 批次面经

## 基本情况

## 流程时间线

## 技术问题

## 项目追问

## HR 问题

## 复盘建议
```

## 四、风险提醒

如果不确定内容是否能公开，默认不要公开。面经的价值在于问题脉络和复盘方法，不在于泄露具体题库或内部材料。
