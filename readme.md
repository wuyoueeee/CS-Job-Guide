## 从这里开始

CS Job Guide 是一份面向计算机专业学生的求职资料库，适合正在准备实习、春招、秋招和职业选择的同学。这里不追求资料越多越好，更希望帮你在正确的阶段找到真正需要的内容。

```mermaid
flowchart LR

    D --> E[模拟面试与复盘]
    E --> F[持续投递]
    F --> G[比较 offer]
    G --> H[完成签约]
```
